# ポケモン親善島Bot-DJS_v13　[![Discord](https://img.shields.io/badge/Discord-Join-blue?logo=discord&logoColor=white)](https://discord.com/servers/pokemonqin-shan-dao-1039799815996968970)
ポケモン親善島で使用しているBotのソースコードです。(一時的に公開しています。)

## Discord v13
DJSv13のソースコードです。
DJSv14は[こちら](https://github.com/roto250/Pokemon-Sinzentou-Bot)
↑機能が多少異なります。

## 動作環境
・Node.js 18 以上<br>・discord.js v13<br>・axios
```
npm init -y
npm install discord.js@v13 axios
```
